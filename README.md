# Tobakksbutikk Admin Panel

## Kjør lokalt
```bash
npm install
npm run dev
```

Åpne:
http://localhost:3000

Admin:
http://localhost:3000/admin

CSV-import støtter:
sku,name,brand,category,price_nok,stock,description

Merk: Dette er første admin-versjon med localStorage. Neste steg er Supabase database + ekte login.
